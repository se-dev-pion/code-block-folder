# code-block-folder

## Introduction

***A simple Visual Studio Code extension enabling customized code folding with comments to make it easier to read codes.***

## Supported Languages

- `c`
- `c++`
- `c#`
- `java`
- `kotlin`
- `dart`
- `swift`
- `javascript`
- `typescript`
- `go`
- `python`
- `ruby`
- `shell`
- `php`
- `perl`
- `rust`
- `lua`
- `sql`
